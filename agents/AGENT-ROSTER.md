## Usage Guide

1. Pick the agent that matches your current task
2. Paste its activation prompt into Claude Code
3. Include the CLAUDE.md context by starting with: `Read CLAUDE.md first, then activate as [AGENT NAME]`
4. One agent per session for best results
5. For complex tasks spanning multiple agents, use one agent per session and hand off via feature specs
